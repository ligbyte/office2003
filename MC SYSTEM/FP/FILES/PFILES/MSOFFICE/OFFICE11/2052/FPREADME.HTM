FrontPage 每次启动时间比较长

当你运行 FrontPage 时，可能会遇到 FrontPage 每次启动的时间都比较长的问题。解决的方法是，在 FrontPage 中点击“工具->选项”菜单，在弹出的“选项”对话框中清除“常规”选项卡中的“检查 FrontPage 是否为网页的默认编辑器”选项，关闭对话框。这样，以后再启动 FrontPage 时就不会遇到这个问题了。

Internet Explorer 增强的安全配置问题

默认情况下，安装 Microsoft Windows Server 2003 时将会启用 Microsoft Internet Explorer 增强的安全配置。这些设置可以阻止某些网站 — 包括使用 Microsoft Office FrontPage 2003 创建的网站 — 按预期效果正确显示或执行。
