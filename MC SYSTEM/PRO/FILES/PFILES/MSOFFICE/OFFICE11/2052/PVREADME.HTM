关于 PowerPoint 播放器

Microsoft Office PowerPoint Viewer 使您可查看在 Microsoft Office PowerPoint 2003 中所创建的包含全部功能的演示文稿。PowerPoint 播放器支持查看在 PowerPoint 97、PowerPoint 2000 和 PowerPoint 2002 中所创建的演示文稿。 PowerPoint 播放器支持打开受密码保护的 PowerPoint 演示文稿。

系统要求

  • 配置为 200 MHz 或更高处理器的 PC
  • Microsoft Internet Explorer 5 或更高版本
  • 被支持的操作系统：Microsoft Windows 98 Second Edition、Windows Millennium Edition、Windows 2000 (Service Pack 3)、Windows XP 或 Windows Server 2003。
  • 随机存取内存 (RAM)：
    • 对于 Windows 98 SE 和 Windows ME：32 MB RAM
    • 对于 Windows 2000 (Service Pack 3)：64 MB RAM
    • 对于 Windows XP 和 Windows Server 2003：128 MB RAM
  • 5 MB 可用硬盘空间
  • 超级 VGA (800 x 600) 或更高屏幕分辨率

多语言问题

PowerPoint 播放器不支持 Microsoft Office 校对工具。

PowerPoint 播放器不支持在 PowerPoint 97 中所创建的双语言演示文稿。

使用 PowerPoint 播放器

PowerPoint 播放器不能正确显示图示动画。如果不显示动画，则图示将完全显示。

PowerPoint 播放器不支持打开信息权限管理 (IRM) 演示文稿。

PowerPoint 播放器不支持运行宏、程序或打开已链接或嵌入的对象。

PowerPoint 播放器不支持显示演示文稿中所链接的这些图形格式：.eps、.pct、.emz、.cdr、.cgm、.wpg 和 .wmz。
