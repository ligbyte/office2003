兼容性问题

InfoPath 2003 需要 Microsoft Internet Explorer 6.0

InfoPath 2003 需要 Microsoft Internet Explorer 6.0 或更高版本。若要下载 Internet Explorer 6，请前往 Internet Explorer 网站。

更新 Internet Explorer 可以提高稳定性

如果 InfoPath 2003 经常停止响应，请尝试更新 Internet Explorer 6。

若要查找 Internet Explore 6 的所有修补程序，包括最新的累积修补程序，请前往 Internet Explorer 下载中心页面。

InfoPath 不支持部分语言的文字输入

InfoPath 2003 不支持印度-伊朗语族的印度语支、南亚或从右到左语言的文字输入。如果您尝试使用这些语言填充表单，InfoPath 可能显示错误信息或停止响应。

在表单中查找和替换文字

清除通配符复选框可能在全字匹配搜索时遇到意外结果

在 InfoPath 中，您可使用“查找”和“替换”任务窗格中的选项在表单中搜索指定的词或短语。如果您选中并清除“使用通配符”复选框，然后使用“全字匹配”选项搜索整个词，InfoPath 不会按照所需的方式工作。（例如，如果您搜索“enter”，InfoPath 除了搜索“enter”，还会搜索“entertain”。）若要解决该问题，请重新启动 InfoPath。

在启用日语时使用全字匹配搜索

如果您启用了支持日语语言功能，然后在表单中搜索非日语的文字，则您必须选中“区分全/半角”复选框，以便“全字匹配”选项正确工作。“查找”和“替换”任务窗格都包含这两个选项。
