安装

“完全安装”会引起繁体中文最终用户定义字符的显示问题

如果使用繁体中文字体的用户在 Office 安装过程中选择“完全安装”选项，或在自定义安装过程中选择“从本机运行”选项，用户所定义的任何最终用户定义字符 (EUDC) 都将被一种朝鲜语字体 (New Gulim) 字符替代。EUDC 将不再可用，无法切换回原来的字体。若要避免此问题，请在安装 Office 时采用自定义安装，并选择“首次使用时安装”选项。如果已安装 Office，请卸载该朝鲜语字体以解决此问题。

常规使用问题

传真服务无法发送 TIFF 文档附件中的墨迹注释

如果直接从 Microsoft Office Document Imaging 发送传真，墨迹内容将正常发送。但是，如果保存文档作为附件使用，则必须确保选中“在保存文档时使注释永久化”复选框（“选项”对话框，“注释”选项卡）。

发送彩色文档时必须包含封面

如果试图发送没有封面的包含彩色对象的 Word、Excel 或 PowerPoint 文档，传真服务提供程序将不会识别此类附件。而如果发送时包括封面，传真服务提供程序就可以发送此类文档。

Office 2003 中的“日积月累”内容没有更新

如果选择使用“Office 助手”并启用“启动时显示‘日积月累’”选项，所看的提示内容可能会是过时的信息。

升级到 Microsoft Office 2003 时没有更新产品和文件图标

在“开始”菜单和 Microsoft Windows 资源管理器中，Microsoft Office 2003 系统应用程序和文件一般以高色彩图标显示。如果从 Microsoft Office 的早期版本升级，则看到的将是从旧版本继承的低色彩图标，而不是 Microsoft Office 2003 所带的新图标。通过更改系统颜色深度，再将其更改回原始设置（步骤如下所示），可确保图标显示正常。

  1. 在“开始”菜单上，单击“控制面板”。
  2. 单击“外观和主题”。（如果看不到此选项，请单击“切换到分类视图”。）
  3. 单击“更改屏幕分辨率”。
  4. 记下“颜色质量”的当前设置。
  5. 在“颜色质量”下，选择一个不同的设置。如果可能，请在该列表中选择更高的设置。
  6. 单击“应用”。
  7. 在“颜色质量”下，选择原始设置。
  8. 单击“确定”。

如果没有安装 SP-1，Office 与 Windows XP 主题不相符

Microsoft Office 2003 应用程序针对 Windows XP 的外观而设计。必须安装 Windows XP Service Pack 1 (SP-1)，Microsoft Office 2003 才可充分利用 Windows XP 的主题。

请访问 Microsoft Windows Update 以下载和安装 Windows XP 最新的 Service Pack。

登录到 Extranet 文档工作区要求有通过邮件发送的用户名和密码，并且 Extranet 网站必须是一个可靠的网站

如果想登录到一个 Windows 网络之外的文档动作区网站，您必须使用通过电子邮件发送的用户名和密码 — 不是您的 Windows 用户名和密码。

同时，要访问此文档工作区，您还必须在 Microsoft Internet Explorer 中将其加入到受信任的网站列表中。

可编程性

在 MSDN 上查找 MODI VBA 参考信息

Microsoft Office Document Imaging 2003 (MODI) 应用程序的功能可以通过可编程对象进行自定义设计。但是，安装 Microsoft Office 2003 时所安装的“帮助”文件中并不包含 MODI 对象模型的文档资料。编程人员可方便地将 MODI 对象库添加到自己的项目中，以将其他应用程序中的 MODI 对象用作自定义应用程序的一部分。也可在 MDSN 上获取 MODI 对象文档资料。

  1. 在“工具”菜单中，选择“引用”以显示“引用”对话框。
  2. 向下滚动找到“Microsoft Office Document Imaging 11.0 Type Library”。
  3. 选中该 MODI 库对应的复选框，再单击“确定”。

此时，您的 Visual Basic 项目就会引用应用程序的对象库。如果打开“对象浏览器”（按 F2）并选择应用程序库，将显示所选对象库提供的对象以及每种对象的方法和属性。在“对象浏览器”中，可在“类”框中选择类并在“成员”框中选择方法或属性。使用复制和粘贴操作可向代码中添加语法。

  1. 连接到 Microsoft Developer Network (MSDN) Library。
  2. 在导航窗格中，展开“Office Solutions Development”节点。
  3. 展开“Microsoft Office Document Imaging”项。
