升级问题

从 Microsoft Office XP 升级后，语法、辞典和断字 .lex 文件可能丢失

如果用户在未安装 Microsoft Windows XP 或 Microsoft Windows 2000 的最新 Service Pack（Windows XP SP1 或 Windows 2000 SP4）的情况下从 Microsoft Office XP 升级到 Microsoft Office 2003，然后访问语法功能，则语法、辞典和断字 .lex 文件可能丢失。例如，在（美国）英语版中，这些文件为：语法 (msgr3en.lex)，断字 (mshy2_en.lex) 和辞典 (msth3am.lex, msth3br.lex)。

若要避免该问题，请使用“工具”菜单上的“拼写和语法”命令访问语法功能。使用该命令会安装所需的文件。
