常规使用问题

无法编辑页选项卡

您无法编辑或重命名页选项卡上的文字。例如，当您单击或右键单击页选项卡时，不会看到重命名或编辑文字的选项。这是由于页选项卡上包含的文字来自于页标题的标题文本或页面上的第一行文字。

若要更改页选项卡上的文字，请编辑页标题中的标题或页面上的第一行文字。

通知区域中的 OneNote 图标丢失

如果您在计算机中删除或重新安装 OneNote，或从任务栏最右侧的通知区域中删除了 OneNote 图标（右键单击“打开新便笺”，然后单击“删除 OneNote 图标”）则图标不再显示在通知区域中。若要在通知区域中显示图标，请依照下列步骤操作：

  1. 启动 OneNote，然后在“工具”菜单上单击“选项”。
  2. 单击“其他”类别。
  3. 选中“将 OneNote 图标放在任务栏中的通知区域”复选框。
  4. 单击“确定”。

OneNote 在“Windows 任务管理器”中作为运行的程序列出

在“Windows 任务管理器”中，“Microsoft Office OneNote 2003 - Windows 任务栏”作为正在运行的程序列出在“应用程序”选项卡上。即使您未启动 OneNote 或 OneNote 未处于运行状态时也可能出现该情况。

如果在 OneNote 中选中了“将 OneNote 图标放在任务栏中的通知区域”复选框，则在后台运行便笺的快速启动功能。启动 Microsoft Windows 操作系统时，会同时启动便笺功能。

若要关闭快速启动功能并从“Windows 任务管理器”正在运行的程序列表中删除 OneNote，请在通知区域中右键单击“打开新便笺”，然后单击“删除 OneNote 图标”。

若要恢复快速启动功能，请依照下列步骤操作：

  1. 启动 OneNote，然后在“工具”菜单上单击“选项”。
  2. 单击“其他”类别。
  3. 选中“将 OneNote 图标放在任务栏中的通知区域”复选框。
  4. 单击“确定”。

重要信息 若要使便笺的快速启动功能正确工作，需要“Microsoft Office OneNote 2003 快速启动”快捷方式。如果该快捷方式从“启动”文件夹中丢失，请按照下列步骤运行“检测并修复”：

  1. 启动 OneNote，然后在“帮助”菜单上单击“检测并修复”。
  2. 在“检测并修复”对话框中，选中“修复时恢复快捷方式”复选框，然后单击“开始”。

文件锁定问题

MSN 上存储的 OneNote 文件可以同时被多台计算机编辑

如果您打开 MSN 上存储的分区（在“文件”菜单上，单击“打开”，单击“网上邻居”，然后单击“MSN 上的‘我的网站’”），在您进行编辑时分区处于未被锁定的状态。 这意味着其他具有文件编辑权限的用户可以打开该分区并在您编辑的同时对其进行更改。任何用户在将其更改保存到 MSN 时都会覆盖其他用户所做的更改。这是 MSN 服务器的限制，没有解决办法。

此外，如果您与 MSN 上的其他用户合作编辑文件，则强烈建议您授予这些用户 Assistant Manager 或更高权限。如果具有 Assistant Manager 更低权限的用户打开一个文件，则该用户会看到文件可以编辑，但所做的更改仅保存在该用户的计算机上，而无法编辑 MSN 服务器上的原始文件。

输入法编辑器必须关闭，才能在多台计算机上编辑文件

如果您打开的节已经在另一台计算机上打开，并且该计算机上的输入法编辑器处于打开状态，则文件处于锁定，无法编辑。要解决此问题，请在第一台计算机上执行以下操作:

  • 关闭 IME。
  • 关闭文件。
  • 退出 Quit OneNote。

手写识别问题

手写文字和 Microsoft Windows XP Tablet PC 的语言版本必须匹配

若要使手写文字搜索和手写文字转换正常工作，则手写文字和所使用的 Microsoft Windows XP Tablet PC 版本的语言 ID 必须匹配。即使某种语言的将墨迹转换为文本的操作无法工作，转换命令仍可用。

输入法编辑器 (IME) 问题

使用 Microsoft IME 的当前版本

如果您使用 Microsoft IME 2000 及其更早版本或第三方的输入法编辑器，则数据输入可能出现问题。为了得到最佳使用效果，请使用 Microsoft IME 2002 或更高版本。

打印问题

“打印”对话框中的部分选项不可用

如果您单击“文件”菜单上的“打印”，则部分打印范围选项可能不可用。“打印”对话框中的打印范围选项由您正在编辑的文档的特性决定，这些选项如下：

  • 全部 如果选中该选项，则打印整篇文档。在默认情况下，如果文档仅包含一页，则自动选中该选项。如果文档包含两个或更多页面，则默认情况下选中“页码”选项。
  • 选定范围 如果您选中该选项，则打印当前所选的页面。如果您未选中任何页选项卡，则该选项不可用。若要确认已选中页选项卡，请单击页选项卡，指向“编辑”菜单上的“选定”，然后单击“页”。
  • 当前页面 该选项在 OneNote 中不可用。（在大多数 Office 程序中，该选项打印插入点所在的页面。如果您选择超过一个页面，则程序会打印所选内容起始处的页面。）
  • 页码 如果您选中该选项，则打印包含插入点的页面或包含所选文字的页面。您也可在“页码”框中键入页码范围（例如，键入“1-3”打印第 1 页至第 3 页）。

“打印预览”设置未被保存以用于后续的 OneNote 会话

如果您在“打印预览”对话框中进行了更改，其中部分设置会应用于在同一个 OneNote 会话中打印的其他分区或页面。当您退出 OneNote 后，不会保存“打印预览”设置的更改。

对下列设置的更改应用于在一个 OneNote 会话中打印的所有页面：

  • 打印页标题或单独的页面
  • 调整内容以适应页宽
  • 页脚
  • 从 1 开始页面编号

注意 对下列设置的更改仅应用于当前被打印的页面：“打印范围”、“纸张大小”和“方向”。

项目符号和编号格式问题

电子邮件或发布的页面中的编号格式发生改变

如果您作为电子邮件发送页面或作为单个文件网页发布页面，则在浏览器或电子邮件中浏览页面时，编号列表的编号格式可能发生改变。发生此问题是因为 OneNote 在发布的页面或发送的电子邮件中仅支持“超文本标记语言”(HTML) 的列表格式。若要解决该问题，请在创建用于发布或通过电子邮件发送的笔记时，仅使用 HTML 列表格式。OneNote 支持下列 HTML 列表格式：

  • 1., 2., 3.,
  • a., b., c.,
  • i., ii., iii.,
  • I., II., III.,
  • A., B., C.,

电子邮件或发布的页面中手写输入的项目符号和编号格式可能无法正确显示

如果您作为电子邮件发送页面或作为单个文件网页发布页面，具有项目符号或编号格式的手写输入内容可能无法显示正确的格式。如果您发布或发送手写输入的内容，则最好不要对其应用项目符号或编号列表。

复制和粘贴问题

在 OneNote 中粘贴的 HTML 表格显示为包含一列的表格

如果您将网页中的表格粘贴到 OneNote 中，则表格格式可能变为单列的表格。OneNote 的设计功能是使用水平大纲级别模拟表格结构，而不保留典型的 HTML 表格格式。

注意 粘贴表格时，无论在“粘贴选项”按钮菜单上选择什么粘贴选项，都可能出现该问题。

若要解决该问题并保留表格列的格式，请将表格粘贴到 Microsoft Office Word 2003 文档中，然后从 Word 将表格复制并粘贴到 OneNote。

电子邮件问题

OneNote 的电子邮件功能需要 Microsoft Office Outlook 2003

如果您使用 Microsoft Office Outlook 2003 作为电子邮件程序，则在 OneNote 中没有发送邮件的命令或按钮。另外，如果您未安装 Outlook 2003，则无法在 OneNote 中创建 Outlook 任务。

如果您希望使用电子邮件将笔记发送给使用除 Outlook 2003 外其他程序的用户，则必须保存笔记并将文件作为电子邮件的附件，步骤如下：

  1. 在“文件”菜单上，单击“另存为”，在计算机上保存 OneNote 文件。该操作会将整个分区保存为 .one 文件。

    注意 如果仅需要发送分区中的一个页面，请单击“文件”菜单上的“发布”。您可以将页面保存为 .mht 或 .one 文件。只有 OneNote 用户可以打开 .one 文件，但使用 Web 浏览器的任何用户都可以打开 .mht 文件。

  2. 在电子邮件程序中将文件为新电子邮件的附件，然后发送邮件。
